# Rock Paper Scissors (CLI)

   This is my **first Python mini-project** - a simple command-line Rock, Paper, Scissors game built while learning Python fundamentals.

   # What it does
   - Play against the computer
   - Accepts input as a number (1/2/3) or a word (rock/paper/scissor)
   - Handles invalid input gracefully

   # What I learned
   - Using the `random` module
   - Dictionaries for mapping values
   - Exception handling with try/except

   # Run it
```bash
   python Rock_paper_scissor.py
```
